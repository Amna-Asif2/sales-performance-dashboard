# Sales Performance Dashboard (Beginner Project)

## Project Overview
This project analyzes sales performance using historical sales data to identify trends, top-performing products, and regional sales distribution.

The goal was to practice **data cleaning, Excel analysis, pivot tables, and dashboard creation**.

## Tools Used
- Excel (Data Cleaning, Formulas, Pivot Tables)
- Tableau (Dashboard Visualization)
- GitHub (Project Showcase)

## Dataset
Source: Kaggle  
Sample Sales Data

## Project Workflow

### Step 1: Data Cleaning
- Removed empty rows
- Converted order dates to proper date format
- Ensured numeric consistency for sales and quantity columns

### Step 2: Key Metrics (Excel)
- Total Sales
- Average Order Value
- Total Orders
  
### Step 3: Pivot Table Analysis
- Sales by Year
- Sales by Product Line
- Sales by Country

### Step 4: Dashboard Creation
- Monthly Sales Trend (Line Chart)
- Top Products (Bar Chart)
- Sales by Region (Map)

### Step 5: Insights
- Classic Cars generated the highest revenue
- Sales peak in November indicating strong year-end demand
- North America leads in overall sales performance
